Rencana integrasi konten PDF ke web desa (Dusun Jetis)

Tujuan
- Memetakan isi PDF ke halaman/section web yang ada
- Menandai konten yang belum ada di web saat ini
- Menyusun urutan implementasi agar perubahan hanya "ubah dikit" di struktur yang sudah ada

Ringkasan isi PDF (high-level)
- Profil umum: kata pengantar, identitas buku
- Struktur organisasi dusun
- Profil kepala dusun (Eka Wulanjari)
- Visi & misi dusun
- Sejarah dusun
- Geografis/denah wilayah
- Sarana/prasarana (wilkel, buk pengairan peninggalan Belanda, patung KB, gudang ketahanan pangan)
- Potensi ekonomi (bengkel andong, sentra bibit jamur, sentra jamu, penggilingan padi)
- Lembaga pemberdayaan masyarakat (Pokgiat, PKK, Posyandu, Karang Taruna, Dasawisma, Takmir Masjid, Poktan, Jaga Warga)
- Kegiatan dusun (lomba anak, mitigasi bencana, edukasi keuangan, KWT, film edukatif, pelestarian mainan tradisional, pemilahan sampah, gotong royong, digitalisasi data penduduk, pembuatan akun sosmed/website, 17-an, syawalan, ramadhan, fun learning, ecoprint)

Kondisi web sekarang (yang sudah ada)
- Homepage: hero, profil singkat, potensi carousel, galeri, kontak
- Halaman Profil: sejarah, populasi, potensi unggulan, batas desa + peta, visi misi, jam operasional
- Halaman Galeri: kumpulan foto tematik
- Halaman Kontak: alamat, email, jam operasional, petugas WA

Gap utama (belum ada di web)
- Struktur organisasi dusun (bagan dan daftar fungsi)
- Profil kepala dusun (bio singkat, periode jabatan, visi personal)
- Sarana/prasarana dusun (wilkel, buk pengairan, patung KB, gudang pangan)
- Lembaga kemasyarakatan lengkap (Pokgiat, PKK, Posyandu, Karang Taruna, Dasawisma, Takmir, Poktan, Jaga Warga)
- Kegiatan dusun yang beragam (event tahunan dan program mahasiswa)
- Geografis/denah wilayah (peta sederhana + keterangan fasilitas)
- Potensi ekonomi spesifik (bengkel andong, bibit jamur, jamu, penggilingan padi)

Catatan penting
- PDF membahas Dusun Jetis (Patalan, Bantul), sementara web sekarang menyebut Desa Segarjaya (Karawang). Semua teks dan label perlu diselaraskan ke Dusun Jetis.

Pemetaan konten PDF -> halaman/section yang cocok
- Profil (halaman Profil)
  - Sejarah dusun -> section Sejarah
  - Visi & misi -> section Visi Misi (ganti teks)
  - Struktur organisasi -> section baru
  - Profil kepala dusun -> section baru
  - Geografis/denah -> section baru (atau ganti peta)
  - Sarana/prasarana -> section baru
- Potensi (bisa di halaman Profil atau halaman Potensi khusus)
  - Bengkel andong
  - Sentra bibit jamur
  - Sentra jamu
  - Penggilingan padi
- Lembaga & Organisasi (halaman baru atau section di Profil)
  - Pokgiat, PKK, Posyandu, Karang Taruna, Dasawisma, Takmir, Poktan, Jaga Warga
- Kegiatan dusun (halaman baru atau section di Profil)
  - Event tahunan (17-an, syawalan, ramadhan)
  - Program pemberdayaan (mitigasi bencana, KWT, pemilahan sampah, gotong royong, digitalisasi data, sosmed/website)
  - Kegiatan anak/edukasi (fun learning, ecoprint, film edukatif, lomba)

Prioritas eksekusi (agar perubahan cepat)
1) Ganti semua label dan teks inti Segarjaya -> Dusun Jetis (nama, lokasi, narasi utama)
2) Update Profil page: sejarah, visi-misi, potensi unggulan, peta/denah
3) Tambah section Struktur Organisasi + Profil Kepala Dusun
4) Tambah section Sarana/Prasarana
5) Tambah section Lembaga Kemasyarakatan (grid card)
6) Tambah section Kegiatan Dusun (timeline/accordion)
7) Update galeri jika ada foto dari PDF (opsional)

Data yang perlu disiapkan (supaya layout cepat diisi)
- Nama dusun lengkap, lokasi administratif
- Foto kepala dusun + biodata singkat
- Bagan struktur organisasi (bisa versi teks dulu)
- Daftar sarana/prasarana + deskripsi singkat
- Daftar lembaga + ketua + jadwal kegiatan
- Daftar kegiatan dusun + ringkasan 1-2 kalimat
- Foto-foto pendukung per kategori

Output akhir (yang diharapkan)
- Halaman Profil lengkap berisi sejarah, visi misi, struktur organisasi, kepala dusun, geografis, sarpras
- Halaman Potensi (atau section) berisi potensi ekonomi spesifik
- Halaman Lembaga Kemasyarakatan
- Halaman Kegiatan Dusun
- Konten homepage diselaraskan ke Dusun Jetis
